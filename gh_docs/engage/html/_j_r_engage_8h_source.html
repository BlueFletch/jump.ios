64 /* Preprocessor directive that conditionally compiles the code that uses the weakly-linked MessageUI.Framework.
65  This framework is required if you want to include the ability to share activities with email or SMS. By default
66  the JRENGAGE_INCLUDE_EMAIL_SMS flag should always be set to "1", which can cause errors with the linker if the
67  framework isn't included, but it's figured most apps would want the email and SMS sharing ability. If you don't want
68  to add the MessageUI.Framework to one or more of your apps that use the JREngage library, you don't have to change
69  this value to "0" (which could cause merging issues if I make changes later).
70 
71  Instead, if you want to override this setting, you need to add the preprocessor flag "OVERRIDE_INCLUDE_EMAIL_SMS"
72  to your target's GCC_PREPROCESSOR_DEFINITIONS build setting (also listed as "Preprocessor Macros" under the
73  "GCC 4.2 - Preprocessing" heading). */
74 #ifndef OVERRIDE_INCLUDE_EMAIL_SMS
75 #define JRENGAGE_INCLUDE_EMAIL_SMS 1
76 #endif
