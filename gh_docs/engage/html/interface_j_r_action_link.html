Detailed Description

A link a user can use to take action on an activity update on the provider.

Create an action link object, fill in the object's fields, and add the object the JRActivityObject.action_links array of your JRActivityObject.

Each action link must contain an NSString* representation of a link, href, and some text, describing what action will happen if someone clicks the link. The href link must be well-formed; that is, it must have both a scheme and host, and conform to RFC 2396.

Example:
 action_links:
 [
   {
     "text": "Rate this quiz result",
     "href": "http://example.com/quiz/12345/result/6789/rate"
   },
   {
     "text": "Take this quiz",
     "href": "http://example.com/quiz/12345/take"
   }
 ]
