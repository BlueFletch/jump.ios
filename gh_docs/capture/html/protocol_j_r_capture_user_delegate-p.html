Detailed Description

Protocol adopted by an object that wishes wishes to to receive notifications when creating a JRCaptureUser on the Capture server or fetching the remote JRCaptureUser from the Capture server.

Parameters
fetchedUserThe JRCaptureUser that was fetched from the Capture server
contextThe same NSObject that was sent to the method JRCaptureUser::fetchCaptureUserFromServerForDelegate:context:(). The context argument is used if you would like to send some data through the asynchronous network call back to your delegate, or nil. This object will be passed back to your JRCaptureUserDelegate as is. Contexts are used across most of the asynchronous Capture methods to facilitate correlation of the response messages with the calling code.
